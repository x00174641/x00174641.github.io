To-Do List Web Application System

This project was developed by myself! This application lets users create their to-do list schedule so they can orgainise themselves better! Functions include, what do to do next ( filtering by the date due and time ) with a live countdown timer displaying how long they got, Marking as complete, Updating the task time due and name and reversing the completed task! When reversing a completed task it checks the due date and time, If the date and time set by the user has gone past then we set a new time for 1 week!
